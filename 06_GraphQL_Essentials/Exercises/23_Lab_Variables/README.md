# 🧪 Lab: Apollo Variables Hello App

Build a minimal full-stack Apollo app where a React component sends a variable to a GraphQL greeting query and renders the result.

**Estimated time:** 7-10 minutes

## 🎯 Goal

Practice using GraphQL query variables from Apollo `useQuery()` in React.

## 📁 Folders

- **Starter/**: Student version with TODOs
- **Solution/**: Completed reference version

## 🧭 Choose Your Path

### Starter Path

1. Open `Starter/README.md`
2. Run `npm run install:all`
3. Run `npm run dev`

### Solution Path

1. Open `Solution/README.md`
2. Run `npm run install:all`
3. Run `npm run dev`

## ✅ Results

By the end of this Lab, students can:

- Define a GraphQL query that accepts variables
- Pass variable values from a component with Apollo `useQuery()`
- Render loading, error, and data states from variable-driven queries
