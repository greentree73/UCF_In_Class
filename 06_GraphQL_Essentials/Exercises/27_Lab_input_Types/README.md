# 🧪 Lab: Apollo Input Type Mutation

Build a minimal full-stack Apollo app where a React form submission runs a GraphQL mutation that accepts an **Input Type** variable and saves data to MongoDB.

**Estimated time:** 7-10 minutes

## 🎯 Goal

Practice Apollo `useMutation()` with an input object variable in a form submission flow.

## 📁 Folders

- **Starter/**: Student version with TODOs
- **Solution/**: Completed reference version

## 🧭 Choose Your Path

### Starter Path

1. Open `Starter/README.md`
2. Run `npm run install:all`
3. Run `npm run dev`

### Solution Path

1. Open `Solution/README.md`
2. Run `npm run install:all`
3. Run `npm run dev`

## ✅ Results

By the end of this Lab, students can:

- Define a mutation that accepts an Input Type variable
- Execute that mutation with `useMutation()` on form submit
- Pass `variables: { input: { ... } }` from React form data
- Persist and return mutation data from a MongoDB-backed resolver
