# Lab Activity: Database Seeding with Tasks

## 🎯 Objective
Practice using Cypress Tasks to connect directly to MongoDB for cleaning and seeding data.

In this lab, students will write two Cypress specs that prove:
- seeded records are returned through GraphQL,
- database cleanup tasks reset state as expected.

## 📁 Structure
- `Starter/` → student implementation with TODO tests.
- `Solution/` → completed reference project with passing specs.

## ⏱️ Estimated Time
30-40 minutes

## 🧠 Skills Practiced
- Registering and using `cy.task()` workflows
- Deterministic test-data setup with MongoDB
- GraphQL API assertions driven by seeded fixtures
- Test isolation via `beforeEach` setup
