# Lab: Authentication & Authorization

## 🎯 Goal
Create and discuss tests for protected GraphQL resolvers using token-based headers.

## 🔧 What This Lab Includes
- `Starter/` includes complete runnable code with `TODO` discussion prompts in the custom command and tests.
- `Solution/` includes the fully completed implementation without discussion prompts.

## ⏱️ Estimated Time
20-30 minutes

## 🧭 Lab Flow
1. Follow `Starter/README.md` step-by-step to discuss command design and auth scenarios.
2. Run auth tests and analyze response differences (`UNAUTHENTICATED`, `FORBIDDEN`, success).
3. Compare with `Solution/` after discussion.
