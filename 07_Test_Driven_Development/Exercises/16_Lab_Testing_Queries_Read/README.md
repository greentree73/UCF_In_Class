# Lab Activity: Testing Queries (Read)

## 🎯 Objective
Implement Cypress tests for GraphQL read operations using live Mongoose data.

Students will write tests for:
- **Find All** (`books` query)
- **Find by ID** (`book(id: ID!)` query)

## 📁 Lab Structure
- `Starter/` → TODO-based student implementation.
- `Solution/` → completed reference implementation.

## ⏱️ Estimated Time
30-40 minutes

## 🧠 Skills Practiced
- DB setup with Cypress tasks
- GraphQL query assertions
- Verifying list and single-record read behavior
- Testing against seeded MongoDB data
