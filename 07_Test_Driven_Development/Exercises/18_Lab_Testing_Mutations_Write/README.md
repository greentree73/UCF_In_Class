# Lab Activity: Testing Mutations (Write)

## 🎯 Objective
Create Cypress tests for GraphQL write operations that verify both:
- GraphQL response correctness
- actual MongoDB persistence state

This lab focuses on:
- `createBook` mutation
- `updateBook` mutation

## 📁 Lab Structure
- `Starter/` → student TODO implementation.
- `Solution/` → completed reference project.

## ⏱️ Estimated Time
35-45 minutes

## 🧠 Skills Practiced
- Using Cypress tasks for DB cleanup and inspection
- Testing mutation responses and persisted values
- Verifying updates across API + database layers
