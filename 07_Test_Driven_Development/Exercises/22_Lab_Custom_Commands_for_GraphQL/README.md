# Lab: Custom Commands for GraphQL

## 🎯 Goal
Practice how to create and use Cypress custom commands for GraphQL testing.

## 🔧 What This Lab Includes
- `Starter/` has complete working code with `TODO` discussion prompts in custom command and test files.
- `Solution/` has the same completed implementation without discussion prompts.

## ⏱️ Estimated Time
20-30 minutes

## 🧭 Lab Flow
1. Use `Starter/README.md` for guided discussion steps.
2. Review `cypress/support/commands.ts` and discuss low-level vs domain-level commands.
3. Review `cypress/e2e/custom-commands-graphql.cy.ts` and discuss how commands improve readability.
4. Compare with `Solution/` after discussion.
